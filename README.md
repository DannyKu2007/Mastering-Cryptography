# cryptography-killerpacks
Encrypt - Decrypt, Assymetric - Symmetric, Encode - Decode, &amp; Hash Pack

## How to use CHINZO-72C & ARABICA2RS
<h3> Encode </h3>
_encode("plaintext")

<h3> Decode </h3>
_decode("cipher")

## How to use AER-256 & ARMON-64
<h3> Encrypt </h3>
completeHash("string", "salt")

<h3> Decrypt </h3>
unHash("cipher", "salt")

## How to use ATOM-128
<h3> Encrypt </h3>
a128e("string")

<h3> Decrypt </h3>
a128("cipher")

## How to use SHA-512
<h3> Hash </h3>
hex_sha512("string")
